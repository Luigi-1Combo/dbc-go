# Meteora DBC Examples in Go

## Overview

This repository contains examples of how to use the Meteora DBC program in Go. Powered by [solana-go](https://github.com/gagliardetto/solana-go).

## Prerequisites

- [Go](https://go.dev/doc/install)

## Usage

```bash
go mod tidy
```

```bash
go run main.go
```
